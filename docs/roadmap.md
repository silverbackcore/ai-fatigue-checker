# Roadmap

AI疲れセルフチェックを、MVP公開版からポートフォリオ・実証用プロトタイプへ育てるための改善予定です。

## v0.1.x: 公開品質の改善

### スマホ表示の最適化

- 質問フォームと結果画面の縦長表示を見直す
- スマホでラジオボタンと保存ボタンを押しやすくする
- 結果画面のグラフ表示が横にはみ出さないようにする

### 診断結果コメントの改善

- スコア帯ごとのコメントをさらに自然にする
- 主な疲労タイプごとの説明を、初見でもわかりやすい表現にする
- 改善アクションの重複を減らし、実行しやすい順に並べる

### フィードバック収集

- Googleフォームなどで利用者の感想を集める
- 「質問は答えやすいか」「結果は納得感があるか」「改善アクションは使えるか」を確認する
- 集めた意見を質問文・結果文・UI改善に反映する

## v0.2.x: 履歴とレポート機能

### 履歴の永続化

- Streamlit Community Cloud上でも履歴を残せるようにする
- Supabase / Google Sheets / PostgreSQL などの外部ストレージを比較する
- 個人情報を扱いすぎない保存設計にする

### PDFレポート出力

- 診断結果を1枚のレポートとして保存できるようにする
- スコア、主なタイプ、改善アクション、注意事項を含める
- 研修や面談で共有しやすい形式にする

## v0.3.x: チーム診断版

### チーム・部署別の傾向分析

- 複数人の回答を集計できるようにする
- 部署別・職種別・利用頻度別の傾向を可視化する
- AI導入後の運用改善や利用ルール策定に使える形にする

### AI利用ルールの強化

- スコア傾向に応じて、より具体的なAI利用ルールを提示する
- 個人向けとチーム向けの提案を分ける
- 医療診断ではない旨の注意文は引き続き明示する
